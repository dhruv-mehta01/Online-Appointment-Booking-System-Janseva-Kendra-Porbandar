# Online Appointment Booking System – Janseva Kendra, Porbandar

## 📌 Project Overview

The **Online Appointment Booking System** is a Java-based application designed to simplify the appointment scheduling process for services provided at **Janseva Kendra, Porbandar**.

The system replaces manual booking with a digital solution where users can book appointments easily and administrators can manage records efficiently.

This project was developed using **Java and JDBC** to demonstrate database connectivity and backend logic implementation.

---

## 🎯 Objectives

* Reduce manual appointment booking work
* Provide a simple digital appointment system
* Store and manage appointment records in a database
* Improve service management efficiency

---

## ⚙️ Technologies Used

* Java
* JDBC (Java Database Connectivity)
* MySQL / Database Management System
* Object-Oriented Programming

---

## ✨ Key Features

* User appointment booking system
* Database storage for appointment records
* Efficient record management
* Simple and user-friendly interface
* Reduced waiting time for services
